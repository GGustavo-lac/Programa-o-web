# 📌 Etec Zona Leste - Site Institucional

![texto alternativo](U)

## 📖 Sobre o projeto

Este projeto consiste na recriação de um site institucional inspirado no da Etec Zona Leste.

O objetivo foi desenvolver um site com múltiplas páginas, navegação funcional e um formulário de contato com processamento de dados.

## ⚙️ Funcionalidades

- Navegação entre páginas (Início, Instituição, Cursos, Calendário, Notícias, Contato e Gestão)
- Layout organizado e responsivo
- Página de contato com formulário
- Processamento de dados utilizando PHP
- Exibição de mensagem de confirmação após envio do formulário

## 📩 Formulário

O formulário permite que o usuário informe:

- Nome  
- Endereço  
- Telefone  
- E-mail  

Após o envio, os dados são processados e exibidos em uma mensagem de retorno.

## 📁 Estrutura

O projeto é composto por páginas separadas que formam o site completo, mantendo organização e facilidade de navegação.

## ▶️ Como executar

1. Colocar os arquivos em um servidor local (XAMPP, WAMP, etc.)
2. Acessar pelo navegador usando o arquivo `index.php`
